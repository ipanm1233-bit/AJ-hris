Manajemen Absensi

Import data sidik jari bulanan dan export laporan kehadiran (termasuk cuti/izin).

1. Import Raw Data Absensi

Unggah file Excel mentah (.xlsx) dari mesin sidik jari Anda. Pastikan kolom sesuai dengan format: NIK Karyawan, Nama Karyawan, Tanggal, Jam Kerja Masuk, dsb.

2. Export Laporan Kehadiran

Tarik laporan absensi yang sudah terkalibrasi dengan database pengajuan cuti/izin/sakit dari sistem HRIS.

Manajemen Absensi

Kelola data kehadiran sidik jari, pengkondisian data cuti/ekspedisi, dan cetak laporan payroll bulanan.

1. Import Raw Data Absensi

Unggah file Excel mentah (.xlsx) dari mesin sidik jari Anda. Pastikan kolom sesuai dengan format: NIK Karyawan, Nama Karyawan, Tanggal, Jam Kerja Masuk, dsb.

2. Export Laporan Kehadiran Terpadu (4 Sheet)

Tarik laporan matriks bulanan komprehensif yang dikalibrasi otomatis dengan data cuti, lembur, dan uang makan ekspedisi.

3. Penarikan Otomatis Mesin Kantor

Hubungkan dan tarik log data absensi secara real-time dari Mesin Fingerprint Kantor yang terhubung via API Gateway.

Super Migrasi — Excel ke Cloud Firestore

Portal HRIS & Operasional CV Andela Jaya

Alat ini membaca seluruh sheet pada berkas Excel Anda, membersihkan nama kolom ke snake_case, mengonversi tipe data secara cerdas (termasuk Smart Date Parser anti "US Date Confusion"), lalu menuliskannya ke Cloud Firestore sesuai skema Portal HRIS.

1. Konfigurasi → 2. Unggah & Pratinjau → 3. Migrasi → 4. Seeder Data Dummy

1. Status Koneksi Firebase

Konfigurasi dibaca otomatis dari js/firebase-config.js. Pastikan Anda sudah mengisi kredensial proyek Firebase pada file tersebut sebelum melanjutkan.

Memeriksa koneksi ke Firestore...

2. Unggah Berkas Excel

Unggah file .xlsx sumber data (mis. Data_Pengajuan.xlsx). Sistem akan otomatis mendeteksi & memetakan setiap sheet ke koleksi Firestore yang sesuai.

Peta Sheet → Koleksi Firestore

3. Jalankan Migrasi

Proses ini akan menulis data ke Firestore secara batch. Jangan tutup halaman selama proses berjalan.

4. Seeder Data Dummy — Modul Baru

Modul-modul berikut belum memiliki sumber data di Excel Anda. Klik tombol di bawah untuk membuat skema koleksi otomatis + data contoh, agar sistem tidak kosong saat pertama kali dijalankan.

  • Rekrutmen (ATS) — rekrutmen_pelamar
  • Gimmick & SOP — gimmick_sop
  • Kalender HR — kalender_hr_events
  • Siklus Karyawan — siklus_karyawan
  • Uang Makan Expedisi — uang_makan_expedisi
  • Formulir ISO tambahan (Lembur, Kasbon) — form_config

Andela Jaya HRIS — Alat internal migrasi data. Jalankan sekali saat setup awal sistem.
